快速人脸风格类比

开发环境: ubuntu16.04 cuda8.0 caffe opencv
技术手段: GPU编程，cuda编程，图像算法，矩阵运算
主要内容: 将2张人脸图像（任意人脸A和参考风格B）送入VGG网络，获得其特征图（feature maps）；在特征图的语义对应区域执行patch-match算法，从而获得B到A的映射。利用该算法实现了众多人脸相关应用，包括换脸/美妆/风格化等等。该科研成果发表在人脸合成领域顶级会议FG2019。
亮点难点： 整个项目涉及deep-patch-match、普式变换、WLS滤波、数值计算、超大稀疏矩阵分解等传统图像、工程矩阵，深度学习、cuda编程等众多领域的问题；需要实现者掌握众多领域的知识，并且具有很好的编程实现能力、逻辑能力、数学基础。

  • 时间: 2018年6月-2018年10月
  • 成果: FG接收论文第17号

快速语义风格类比

开发环境: ubuntu16.04 cuda8.0 caffe opencv
技术手段: GPU编程，cuda编程，图像算法，矩阵运算
主要内容: 这是对《快速人脸风格类比》的进一步深入研究，我们希望把快速的语义风格类比从人脸领域推广到任意图像。人脸领域还有landmarks来帮助patch-match的加速与校准，我们急需一种技术手段作为任意图像的landmarks。我们创造性的提出了“语义超像素”这一概念，我们认为：把传统的超像素聚类运用在feature maps上面，就可以得到按照语义聚类的“语义超像素”，我们把相同语义的超像素当做是相同位置的landmarks，实现了对patch-match的加速。我们还提出了一种基于密集匹配的颜色迁移算法，可以在对应的语义区域实现精致的颜色迁移。 除此以外，我们还采用了kd-tree算法和pca降维。
亮点难点： 我们提出了“语义超像素”和语义颜色迁移两种重要的技术手段，解决了任意图像的快速语义风格类比。该项目依然全部使用cuda和c++写成，对实现者的编程能力要求很高；其中众多idea也是天马行空，要求提出者有丰富的想象力

  • 时间: 2018年11月-2019年1月

司法案件便捷管理

开发环境: windows7 java swing
技术手段: java-swing界面 java-webdriver爬虫 数据库 加密 TCP协议
主要内容: 这是我在大学期间自己做的一个项目，幸运的通过这个项目赚取了研究生的学费。当地司法机关使用了某些外包公司的网页管理案件的系统，但是其系统操作繁琐，理解困难；由于该系统是在网页上操作的，所以我得以通过自己写一个java爬虫和简单易懂的界面，自动的实现一系列繁琐的程序化操作（比如案件接收-案件判决-执行人财产锁定）。除此以外，我自己写了通信规则，使用tcp协议来管理和监控同一个单位内部的所有使用者。为了防止软件被盗用，我还自己设计了一套加密手段，每个使用者必须使用由我唯一分发的密钥才能使用。
亮点难点： 在这个项目之前，我从未做过java，只会python爬虫知识。但是3个月内，大四的我完成了这个项目，整个项目设计swing界面设计，java爬虫，数据库管理，tcp通信，文件加密等等众多内容，难度非常大。

  • 时间: 2017年1月-2017年3月
  • 成果: 程序说明文档
